# 運動記録・体重の日次トレンド機能を削り、食事記録(核)に集中する

既存アプリには運動記録(手入力の消費カロリー一覧)と体重タブ(日次体重記録・グラフ)があったが、これらは要件を決めずに追加された機能であり、核である「写真1枚で自動食事記録」への貢献が薄い。

運動記録機能と、体重タブの日次記録・グラフ機能は削除する。ただしプロフィール設定における体重入力(基礎代謝・目標カロリー/PFC自動計算に使う値)は維持する。これは核を支える「残りカロリー」計算に必須のため。

運動記録は写真解析ほど自動化されておらず核とズレる。体重トレンド表示は「続けられる」体験の補助に過ぎず、必要になれば後で改めて要件を決めて追加すればよい。
