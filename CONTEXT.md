# カロミル(食事管理アプリ)

個人用のあすけん風食事管理アプリ。核となる目的は、記録の手間を極限まで減らし「写真1枚で自動記録できる」ことによって、続けられる食事管理を実現すること。体重グラフやPFCバランス表示などはこの核を支える手段であり、目的そのものではない。

## Language

**食事記録(Meal Entry)**:
ユーザーが「食べた」として確定させた1件分の食品データ。日付と食事区分(朝食/昼食/夕食/間食)に紐づき、kcal・PFC・塩分・食物繊維・糖分を持つ。
_Avoid_: 食品、アイテム

**解析結果(Analysis Result)**:
写真からAIが推定した食品名・栄養素の下書き。ユーザーが確認・編集して確定するまでは食事記録にならず、保存もされない。
_Avoid_: 推定値(確定前のものだけを指すときはこちらを使う)
